### ğŸ“ Home Work 4 - CSS Selectors & Specificity

**Assignment:** Practice various CSS selectors and calculate specificity weights.

**Features:**
- 10 different selector examples with detailed descriptions
- Tag selectors (`p`)
- Class selectors (`.highlight`)
- ID selectors (`#main`)
- Attribute selectors (`a[href*="home"]`)
- Contextual selectors (`div span`)
- Child selectors (`div > p`)
- Adjacent sibling selectors (`h2 + p`)
- Selector combinations (`div.container`, `#header .nav`, `div.box > p:first-child`)
- Specificity calculation for each selector
- Color-coded examples for visual differentiation

**Skills Demonstrated:**
- Understanding of CSS selector types
- Specificity calculation (0,0,0,0 format)
- Difference between contextual and child selectors
- Pseudo-classes usage
- Valid HTML and CSS code

**Technologies:** HTML5, CSS3
