# California Electronics Store

Online electronics store website with responsive design.

## Description

A modern e-commerce website for electronics products including laptops, tablets, phones, and watches. Built with mobile-first approach.

## Technologies

- HTML5
- Tailwind CSS v4
- Google Fonts (Inter, Readex Pro)

## Pages

- **index.html** - Homepage with hero banner, categories, search, and newsletter
- **catalog.html** - Product catalog with best sellers
- **product.html** - Product details page

## Live Demo

[View Website](https://alexbirnaz.github.io/Hillel-fullstack-course/home-work-17/)

## Features

- Responsive design (mobile, tablet, desktop)
- Mobile-first approach
- Modern UI/UX design
- Product cards with prices
- Newsletter subscription form
- Search functionality

## How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm run build` to compile Tailwind CSS
4. Open `index.html` in browser
