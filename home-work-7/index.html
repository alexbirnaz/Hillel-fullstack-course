Task 6: Overflow Property

What I learned: Overflow controls what happens when content is too big for container. Visible shows everything (default), hidden cuts off extra content, scroll always shows scrollbars, auto shows scrollbars only when needed. Auto is best choice in most cases.

Task 7: Positioning

What I learned: Position property changes how elements are placed. Static is default (normal flow). Relative moves from original position but keeps space. Absolute positions relative to nearest parent with position (or body). Fixed stays in same place on screen when scrolling. Sticky scrolls normally then sticks to position when reached.

position: absolute - positioned inside parent
position: fixed - always visible in corner
position: sticky - scroll down to see it stick to top
